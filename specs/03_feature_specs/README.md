# Feature Specs

Feature Specs описывают отдельные функции продукта после Global Spec и Functional Map.

## Правила раздела

1. Каждый Feature Spec должен ссылаться на Global Spec и Functional Map.
2. Feature Spec описывает поведение функции, границы, сценарии, состояния и открытые вопросы.
3. Feature Spec не описывает базу данных, API, фреймворки и финальную техническую реализацию.
4. Если функция требует технического решения, оно выносится в Technical Spec.
5. Если функция требует проверки пути пользователя, она выносится в User Stories.
