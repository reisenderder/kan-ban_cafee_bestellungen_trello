# Активные планы разработки

В этой папке хранятся планы, работа по которым еще не завершена или еще не подтверждена пользователем после ручного тестирования.

## Правила

1. Каждый новый план разработки создается в этой папке.
2. План остается здесь на протяжении реализации и проверки.
3. Успешные автоматические тесты сами по себе не завершают план.
4. После явного подтверждения пользователя о завершении ручного тестирования и перед публикацией изменений план перемещается в `../completed/`.
5. При перемещении обновляются статус плана, `../README.md`, все ссылки на файл и соответствующие журналы в `../../roadmap/`.
