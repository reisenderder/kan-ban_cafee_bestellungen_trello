# Завершенные планы разработки

В этой папке хранится история планов, реализация которых завершена и подтверждена пользователем после ручного тестирования.

## Правила

1. План переносится сюда из `../active/` только после явного подтверждения пользователя.
2. Перенос выполняется до `git add`, `commit`, `push` или создания pull request.
3. Одновременно фактически выполненные работы записываются в соответствующие журналы `../../roadmap/`.
4. В перенесенном плане фиксируется завершенный статус.
5. Завершенный план сохраняется как исторический документ и не удаляется.
6. Новая доработка оформляется отдельным активным планом, а не перезаписывает завершенный план.

## Завершенные планы

Завершенных планов пока нет.
